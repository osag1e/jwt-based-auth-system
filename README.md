# jwt-based-auth-system
User Account CRUD with a jwt-based auth system

![logs](https://github.com/OsagieDG/jwt-based-auth-system/blob/main/log.png)


## Makefile commands are:
```
make build-api
make run
make lint
make cyclomatic
make clean
```


## Project environment variables
```
HTTP_LISTEN_ADDRESS=
DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_SSLMODE=
```
